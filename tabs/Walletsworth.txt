Tablature Explanation:

b = bend
r = release
Gb = gradual bend
Gr = gradual release
sb = semi bend (usually 1/4 step to a 1/2 step)
p = pull off
h = hammer on
t = right hand tapping
/ = slide up
\ = slide down
~ = vibrato
x = muted string
ff = 'fretboard fingers' are used to 
     'play' (or pluck) the open strings
     to ring them out instead of the right hand
# = pinch/artificial harmonic
